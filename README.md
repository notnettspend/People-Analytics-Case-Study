# People Analytics Case Study

## Sobre o projeto

Este repositório apresenta um estudo de caso fictício desenvolvido com o objetivo de simular um projeto de People Analytics realizado para um cliente.

O projeto acompanha todas as etapas de um trabalho de análise de dados, desde a definição do problema até a elaboração de um relatório com os resultados obtidos.

Todos os dados utilizados são fictícios e foram criados exclusivamente para fins de estudo e desenvolvimento de portfólio.

## Objetivos

- Aplicar conceitos da linguagem R em um cenário próximo ao mercado de trabalho;
- Desenvolver análises voltadas para People Analytics;
- Documentar todas as etapas do projeto;
- Demonstrar a evolução do projeto conforme novos conhecimentos em R forem sendo adquiridos.

## Estrutura do projeto

- Contexto do projeto;
- Base de dados;
- Desenvolvimento da análise em R;
- Relatório final.

## Tecnologias

- R
- RStudio
