# People Analytics Case Study

Este diretório reúne todas as etapas do estudo de caso desenvolvido ao longo do aprendizado da linguagem **R**, simulando um projeto de consultoria em **People Analytics**.

A proposta é reproduzir o fluxo de trabalho de um projeto real, desde o recebimento das informações da empresa cliente até a documentação das análises e implementação das soluções em código.

Cada etapa representa uma fase do projeto e foi organizada de forma independente, contendo sua própria documentação, arquivos de apoio e scripts desenvolvidos durante a resolução das solicitações apresentadas pela empresa fictícia.

## Organização

O estudo de caso está dividido em módulos, permitindo acompanhar de forma cronológica a evolução do projeto e dos conhecimentos aplicados.

Cada diretório possui:

- Documentação da etapa (`.qmd`);
- Script em R contendo a implementação das atividades;
- Arquivos auxiliares quando necessários;
- README específico descrevendo os objetivos daquela fase.

## Fluxo do Projeto

1. Contexto do Projeto
2. Base de Dados
3. Manipulação de Dados
4. Estruturas Condicionais
5. *(Próximas etapas serão adicionadas conforme a evolução do projeto.)*

## Evolução

Este estudo de caso será expandido continuamente à medida que novos conteúdos da linguagem R forem estudados, mantendo um histórico organizado da evolução do projeto e das técnicas utilizadas durante a construção da solução.
