# Estruturas Condicionais

Esta etapa do projeto é dedicada à aplicação de estruturas condicionais utilizando a linguagem **R** em um contexto de **People Analytics**.

Após a manipulação inicial da base de dados, a equipe de consultoria passa a atender solicitações que exigem tomadas de decisão baseadas em regras de negócio. Para isso, são utilizadas estruturas condicionais que permitem avaliar diferentes cenários e automatizar processos relacionados aos colaboradores da empresa.

As atividades desenvolvidas nesta etapa simulam situações comuns encontradas em projetos reais de Recursos Humanos e People Analytics, como elegibilidade para bônus, promoções, programas internos e classificação de desempenho.

## Objetivos

- Aplicar estruturas condicionais utilizando a linguagem R.
- Desenvolver lógica de programação voltada para regras de negócio.
- Automatizar decisões com base nas informações da base de dados.
- Criar indicadores condicionais para apoio à tomada de decisão.
- Consolidar conceitos fundamentais de programação aplicados ao contexto de People Analytics.

## Conteúdos estudados

- Estrutura `if`
- Estrutura `if...else`
- Função `ifelse()`
- Operadores relacionais
- Operadores lógicos
- Comparações entre variáveis
- Regras de negócio
- Criação de novas variáveis condicionais

## Arquivos

- **01 - Estruturas Condicionais.qmd**  
  Documento contendo a contextualização da etapa, os objetivos e as solicitações encaminhadas pela empresa.

- **02 - Estruturas Condicionais.R**  
  Script em R contendo todas as resoluções das solicitações propostas, utilizando estruturas condicionais.

## Observação

Todas as atividades desta etapa utilizam a base de dados criada anteriormente no projeto, mantendo a continuidade do estudo de caso desenvolvido para a empresa fictícia **Orion Tech Solutions**.
