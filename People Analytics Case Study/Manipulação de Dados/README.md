# Manipulação de Dados

Esta etapa do projeto é dedicada à manipulação da base de dados disponibilizada pela empresa **Orion Tech Solutions**.

Após a construção da base de dados na etapa anterior, o foco passa a ser a exploração e manipulação das informações utilizando a linguagem **R**. As atividades desenvolvidas nesta fase simulam demandas comuns em projetos de People Analytics, permitindo consultar, selecionar, filtrar e resumir informações de acordo com diferentes necessidades do negócio.

Durante esta etapa são aplicados conceitos fundamentais da linguagem R, servindo como preparação para análises mais avançadas que serão desenvolvidas nas próximas fases do projeto.

## Objetivos

- Explorar a estrutura da base de dados.
- Selecionar linhas e colunas específicas.
- Filtrar informações utilizando operadores relacionais e lógicos.
- Aplicar estruturas condicionais.
- Realizar consultas baseadas em regras de negócio.
- Calcular indicadores simples utilizando funções básicas da linguagem.
- Praticar a manipulação de data frames.

## Arquivos

- **01 - Manipulação de Dados.qmd**  
  Documento contendo a contextualização da etapa, as solicitações realizadas pela empresa e os exercícios propostos.

- **02 - Manipulação de Dados.R**  
  Script em R contendo todas as resoluções das solicitações e exercícios utilizando a base de dados construída anteriormente.

## Observação

Todas as atividades desenvolvidas nesta etapa utilizam exclusivamente a base de dados criada no módulo **02 - Base de Dados**, mantendo a continuidade do estudo de caso fictício desenvolvido ao longo deste repositório.
